# تقویم دیواری شمسی ۱۴۰۵

این پروژه یک تقویم دیواری کامل و آماده چاپ برای سال ۱۴۰۵ هجری شمسی است که با LaTeX (XeLaTeX) ساخته شده و شامل ۱۲ صفحه (یک صفحه برای هر ماه) می‌باشد.

## ویژگی‌ها

- تاریخ‌های شمسی ۱۴۰۵ به‌طور دقیق محاسبه شده‌اند (الگوریتم تبدیل جلالی ↔ میلادی).
- تعطیلات رسمی کشور با پس‌زمینهٔ صورتی مایل به قرمز و متن قرمز پررنگ مشخص شده‌اند.
- روزهای جمعه (تعطیل هفتگی) نیز با رنگ قرمز نمایش داده می‌شوند.
- تاریخ میلادی معادل هر روز به‌صورت کوچک زیر عدد روز شمسی آمده است.
- هر صفحه شامل فهرستی از تعطیلات رسمی و مناسبت‌های مهم آن ماه است.
- عکس هر ماه از پوشهٔ `photos` (همان تصاویر قالب اصلی شما) استفاده شده است.
- طراحی با تایپوگرافی راست‌به‌چپ (بومی XeTeX) و بدون نیاز به بسته‌های ناموجود مانند bidi/xepersian.

## ساختار پروژه

```
calendar1405/
├── main.tex          ← فایل نهایی LaTeX (تولیدشده توسط build_tex.py)
├── build_tex.py       ← اسکریپت پایتون که main.tex را می‌سازد (قالب + داده‌ها)
├── gen_data.py        ← داده‌های تقویم: نام ماه‌ها، تعطیلات، مناسبت‌ها، نام فایل عکس‌ها
├── jalali.py          ← الگوریتم تبدیل تاریخ جلالی به میلادی (بدون وابستگی به اینترنت)
├── photos/            ← تصاویر ماه‌ها (از قالب اصلی شما)
└── README.md
```

## نحوهٔ ساخت مجدد PDF

اگر بخواهید مناسبت‌ها یا تعطیلات را ویرایش کنید، فایل `gen_data.py` را باز کرده و دیکشنری‌های
`HOLIDAYS` و `EVENTS` را ویرایش کنید (کلید = روز ماه، مقدار = متن فارسی).
سپس دستورهای زیر را اجرا کنید:

```bash
python3 build_tex.py      # main.tex را از نو می‌سازد
xelatex main.tex          # کامپایل با XeLaTeX (به دلیل نیاز به فونت و راست‌به‌چپ)
xelatex main.tex          # اجرای دوم برای تثبیت رفرنس‌ها (اختیاری)
```

نکته: کامپایل حتماً باید با **xelatex** انجام شود (نه pdflatex)، چون از fontspec
و قابلیت‌های بومی راست‌به‌چپ XeTeX استفاده شده است. فونت استفاده‌شده FreeSerif
است که پوشش کامل حروف فارسی/عربی دارد و از قبل در بسیاری از توزیع‌های TeX Live موجود است.
اگر فونت دیگری (مانند Vazirmatn یا IRANSans) در دسترس دارید، می‌توانید نام آن را
در ابتدای `build_tex.py` در دستور `\setmainfont{FreeSerif}` جایگزین کنید تا ظاهر
زیباتری داشته باشید.

## منبع داده‌های تعطیلات

تعطیلات رسمی و مناسبت‌های سال ۱۴۰۵ بر اساس تقویم رسمی منتشرشده توسط شورای فرهنگ عمومی
کشور و منابع معتبر تقویمی (baheasab.ir، همشهری آنلاین و ...) گردآوری و با محاسبهٔ
مستقل روزهای هفته اعتبارسنجی شده‌اند.
